═══════════════════════════════════════════════════════════
  DEPLOYMENT STEPS
═══════════════════════════════════════════════════════════

Step 1: Copy these 4 files to your GitHub repository
────────────────────────────────────────────────────────────
cd /path/to/your/github/repo
cp app.py .
cp requirements.txt .
cp Dockerfile .
cp mss-logo.png .

Step 2: Commit and push to GitHub
────────────────────────────────────────────────────────────
git add .
git commit -m "Update to v2.0 - Professional PDF template"
git push

Step 3: Deploy on Render.com
────────────────────────────────────────────────────────────
Option A: Automatic (if auto-deploy is enabled)
  → Render will deploy automatically after push

Option B: Manual
  → Go to: https://dashboard.render.com
  → Select: mss-pdf-api
  → Click: "Deploy latest commit"

═══════════════════════════════════════════════════════════
  VERIFICATION
═══════════════════════════════════════════════════════════

After deployment (2-5 minutes):

1. Check API health:
   curl https://mss-pdf-api.onrender.com/health
   
   Should show: "version": "2.0.0"

2. Test PDF export:
   → Open: https://mss-reports-d9f84.web.app
   → Login and open any report
   → Click "Export PDF"
   → Check the new professional format!
